PROJECT TITLE: FlappyRocket
PURPOSE OF PROJECT: Entertainer
VERSION or DATE: v1.0.0 21/Apr/2022
HOW TO START THIS PROJECT: Compile and run the code in the greenfoot software
AUTHORS: G7
USER INSTRUCTIONS:
1. Can be used in windows version
2. Latest version of java sdk and greenfoot are very recommended

HOW TO PLAY
1. Click play button
2. Tap or hold spacebar to make the rocket flap
3. Press 'esc' button to pause the game
4. Click resume button to continue the game
5. Avoid the rocket from being crashed with the poles
6. Score will be counted and the highest score will be recorded
